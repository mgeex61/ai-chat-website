# 🤖 mgeexAI – Moderne KI Chat Website

Eine moderne, minimalistische KI-Chat-Webseite mit echter Gemini API Integration.
Gebaut als Portfolio-Projekt mit Fokus auf Design, UX und echter Funktionalität.

👉 **Live Demo:**
https://ai-chat-website-rgav.vercel.app

---

## 🚀 Features

* 💬 **Echter KI-Chat (Google Gemini API)**
* ⚡ **Streaming-Effekt (Antwort kommt live)**
* 🧠 **Mehrere Chats (wie ChatGPT)**
* 💾 **Chat-Verlauf wird gespeichert (localStorage)**
* 📋 **Copy-Button für Antworten & Code**
* 🧩 **Markdown & Codeblock Support**
* 🎨 **Modernes Dark UI (Apple / Premium Style)**
* 📱 **Responsive Design**

---

## 🛠️ Technologien

* HTML
* CSS (Custom UI, kein Framework)
* JavaScript (Vanilla)
* Vercel (Deployment)
* Google Gemini API

---

## ⚙️ Setup (optional lokal)

Falls du das Projekt lokal starten willst:

1. Repo klonen

```bash
git clone https://github.com/mgeex61/ai-chat-website.git
```

2. API Key setzen (wichtig!)

```env
GEMINI_API_KEY=DEIN_API_KEY
```

3. Projekt starten (z. B. mit Vercel CLI oder lokalem Server)

---

## ⚠️ Hinweis

Der API-Key ist **nicht im Repository enthalten**
und muss als **Environment Variable** gesetzt werden (z. B. in Vercel).
